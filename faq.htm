VASL Frequently Asked Questions

Q: Are there any compatibility issues with Upgrading the board files to the new v6 boards and still running them in VASL 6.0.3?

A: No - they will work just fine in 6.0.3.

Q: Where do I set the server address?

A: VASL determines the default server address dynamically at run-time. In Preferences, there is a Server tab, where there is also a VASSALJabber Server that can be used, and a P2P Network configuration can be set up.

Q: Is the OBA deck randomized in any way regarding its drawing? Or will the same sequence of cards reveal each and every time once a deck is created. What about the "extra" card pulls which are then put back in the deck.

A: The OBA pile randomizes the cards every time the card pile is clicked.

Q: I've seen some crazy results generated by the dice roller in VASL. Can I trust its fairness?

A: Yes. See Stephen Berry's analysis

Q: I have a copy of the BRT map that opens fine in 5.9.2/3.1.18. But when I try to start a New Game in 6.0.3/3.2.11 it doesn't even show up in the list of available maps to choose from.

A: The file name was BdBRT and the fix is to rename it bdBRT.
