Adding markers to the board

You can quickly add markers to the board from the QC toolbar.

The QC toolbar is in the bottom toolbar of the VASL interface.

To add a marker to the board, click on the marker you want to add, and drag it to the board.

If the counter in the marker toolbar has an arrow in the lower right corner, clicking on the marker in the toolbar opens a popup with a number of options. Click in the popup on the marker you want to add to the board and drag it to the board.

  • Creating a new QC bar
    You can create new QC bars for your interface. For example, you might want to create a custom QC bar to use when playing a scenario with Night rules in effect, or for an airborne or amphibous landing.
  • Modifying a QC bar
    You can modify custom QC bars.
  • Activating a QC bar
    Only one QC bar is active at a time. If you want to use a different QC bar, you must activate it.
  • Deleting a QC bar
    You can delete custom QC bars.
