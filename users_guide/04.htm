VASL User's Guide - Environment

Preferences

VASL is composed of several windows. Upon starting, you will see the Server window, the VASL Controls window and the Counters window.

Logging On

In the Server window, click Connect to log on to the VASL server

(Note: Some corporate security firewalls will prevent a connection, although a residential service should have no problem).

Once connected, you should see the following indicators:

  • You will see confirmation in the VASL Controls window chat area that you have connected with the server and entered the Main Room.
  • The Active Games section of the Server window will indicate which players are logged on and which scenarios (or Rooms) exist.
  • If players are chatting amongst each other in the Main Room, text will appear in the VASL Controls chat area.

To Chat

To chat with other players, click in the entry area below the main chat display at the bottom of the VASL Controls window and start typing. Your text will appear in the VASL Controls chat area. To send the message, press Enter on your keyboard.

You can also type while the focus is in the main map window and the typing will be echoed to the chat entry area.

Ctrl-V will paste from the clipboard into the chat entry area.
