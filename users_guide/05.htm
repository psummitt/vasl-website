VASL User's Guide - Getting Started

Joining a Game

Probably one of the best ways to learn the ins and outs of VASL is to spectate a bit.

Step 1 - Enter a Scenario Room

To view a scenario already being played, double-click on the scenario (room) name in the Active Games column of the Server window. To create a new scenario chat room, enter a new name in the "New Game" field and press Enter. A list of players currently in the room you have entered will appear in the "Current Game" column.

Step 2 - Synchronize

With VASL, you are automatically synchronized with the game-in-progress when you enter a new room.

[Rodney, what circumstances would necessitate needing to synchronize from the right-click menu?] Synchronize by right-clicking on the player's name in the Current Game column and selecting Synchronize from the pop-up menu.

Important Note: Once you have synchronized with a game, you may move around the map and inspect either player's counter stacks without interfering. However, do not move the counters or roll the dice, as these actions will be reflected on everyone's game board!

Step 3 - Global View Map

Once synchronized, two new windows will be created. One will be a small Overview window that provides you with a miniature view of the playing area. Within the Overview window you will see an outline of a rectangle. This rectangle shows you the area depicted on the Main Map window. By clicking on a spot within the Overview window you will jump to that spot on the Main Map window.

Step 4 - Main Map Window

The Main Map window is the main playing area of VASL. It is where all the counters are displayed and where the gaming action takes place.

Starting a Game

To start a game of your own, it will be necessary to create a Scenario Room by entering a new name in the "New Game" field of the Server window and pressing Enter. Others (your opponent and spectators) can then enter the room and "synch" in. (See the next section Setting up a Scenario.)

Saving a Game

A game can be saved at any time. In the VASL controls window, simply go to File - Save Game. Provide the scenario with a name and a location on your hard drive, and press OK. It is highly recommended that you get in the habit of saving your games while playing to minimize the inconvenience that would result from a crash. There is no auto-save or back up feature. It is your responsibility to save and protect your games.

Message Boards

VASL also contains a message area that allows you to post messages or game requests on the server. In the Server window, click the "Messages" tab and click "Check Messages" to view the Message Board, or "Post Message" to open the Message Composer window. In the Message Composer window, enter your message and click Send to post your message to the Message Board.

Status

In the Status tab of the Server window, you may indicate that you are looking for a game or that you are away from your keyboard.

Checking the "Looking for game" check box will place an exclamation mark (!) beside your name in the Server window, indicating that you are interested in finding a game.

Clicking the "Away from keyboard" button will place an "X" next to your name in the Server window, indicating that you have stepped away from your computer. A "I'm Back" window will pop up. When you're ready to return, clicking "OK" will close the "I'm Back" window and remove the "X" from beside your name.
