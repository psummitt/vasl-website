Old VASL Scenario Setups

Status Update: January 2024

New versions of these scenario setup files, for 6.6.7 and beyond will now be added to Scenario Setups. This old page will no longer be updated.

This page contains several collections. Most of them are very outdated, one of which included every scenario of the core modules among them - for VASL v4.

Then there is another collection which includes roughly 150 VASL setups stretching from versions v5.9.2 to v6.2.2.

You can also retrieve ASLSK Scenario Setup files at https://github.com/atuline/ASLSK-Scenarios.
