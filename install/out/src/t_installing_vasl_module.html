Installing the VASL module

After you install the VASSAL engine, download and install the VASL module.

To install the VASL module to the VASSAL engine:
  1. Open a browser and go to http://vasl.info/download.htm.
  2. Click the button to download the version of the VASL module you want to install.
    The module is dowloaded to your default donwloads folder.
  3. Move the VASL module (vaslX.X.X.vmod) to a location you can find easily and that VASSAL can access.
    IMPORTANT: Security enhancements to many OS block VASSAL from accessing files under C:\ProgramFiles. Do NOT put VASL under this hierarchy
  4. Start the VASSAL engine.
  5. In the VASSAL menu bar, chooseFile > Open module.
    The VASSAL interface returms a browse dialog.
  6. Navigate to the directory where you placed the vasl module, select the module, and click Open.
    The VASSAL interface returns a dialog stating that the VASSAL engine is tiling the VASL images.
The VASL module is installed to the VASL engine.
