Installing VASL

You can install VASL on the most common operating systems, including Windows, Macintosh, and Linux.

To install VASL:
  1. Install the VASSAL Engine on your computer.
    See Installing the VASSAL engine.
  2. Download and install the VASL VASSAL module on your computer.
    For details, see Installing the VASL module.
  3. Download and install the mapboards and overlays on your computer.
    For details, see Accessing VASL boards and downloading overlay files.
  4. Download and install any plugins you want to use on your computer.
    For details, see Installing VASL extensions.
