VASL Scenario Setups

Status Update January 2024

In conjunction with the inclusion of the Game Updater Functionality in VASL6.6.7 and beyond, this page now includes Scenario Setup files that have been updated to work with the latest VASL version.

These files will be updated each time a new VASL version is released to provide an enduring and up-to-date collection of VASL setups.

All of the previous Scenario Setup files are still available at Scenario Setups.

The current collection of updated Scenario Setups does not include all of the previous file. More will be added over time. In addition, players are encouraged to contribute new Scenario Setup files.
